== Description ==

Nony Portfolio is a modern, glassmorphic WordPress block theme with animated blobs and particles. Perfect for developers, designers, and creatives who want to stand out with a unique portfolio website.

Features:
* Full Site Editing (FSE) support
* Companion plugin for advanced customization
* Glassmorphism design with animated backgrounds
* Responsive mobile navigation with hamburger menu
* Custom admin panel for easy configuration
* Profile badges management
* Social links customization with custom colors
* Blog and page templates included
* Custom 404 error page
* Animated elements and smooth transitions
* Custom context menu
* Mouse parallax effects

== Installation ==

1. Upload the theme folder to /wp-content/themes/
2. Upload the nonylabs-companion plugin to /wp-content/plugins/
3. Activate the theme through the 'Appearance' menu in WordPress
4. Activate the NonyLabs Companion plugin through the 'Plugins' menu
5. Go to NonyLabs > General Settings to customize your site
6. Customize your site using the Site Editor (Appearance > Editor)

For detailed installation instructions, see INSTALLATION.md

== Customization ==

This theme requires the NonyLabs Companion plugin for full functionality:

**NonyLabs Settings:**
* General Settings - Configure navigation logo, site title, footer content
* Profile Badges - Add/edit/remove profile badges with live preview
* Social Links - Manage social media links with custom icons and gradient colors

**WordPress Menus:**
* Go to Appearance > Menus to create custom navigation menus
* Assign menus to "Primary Menu" and "Footer Menu" locations

**Site Editor:**
* Colors and typography in theme.json
* Template layouts in /templates/
* Reusable patterns in /patterns/
* Header and footer in /parts/

== Frequently Asked Questions ==

= Do I need the NonyLabs Companion plugin? =

Yes, the NonyLabs Companion plugin is required for this theme to function properly. It provides the admin interface for customizing navigation, badges, and social links.

= How do I customize the navigation menu? =

Go to Appearance > Menus in WordPress admin, create a new menu, and assign it to the "Primary Menu" location. You can also customize the logo text in NonyLabs > General Settings.

= How do I add or edit profile badges? =

Go to NonyLabs > Profile Badges in the admin dashboard. You can add, edit, or remove badges with a live preview of your changes.

= How do I customize social links? =

Go to NonyLabs > Social Links to manage your social media profiles. You can add custom platforms with Remix Icon classes and customize gradient colors for each card.

= Where can I find icons for social links? =

Visit https://remixicon.com/ to browse thousands of free icons. Copy the icon class (e.g., ri-github-fill) and paste it into the social link settings.

= Can I use this theme without the companion plugin? =

The theme will work, but you won't be able to customize navigation, badges, or social links through the admin panel. The companion plugin is highly recommended.
