Companion plugin for Nony Portfolio theme with advanced customization options.

== Description ==

The Nony Portfolio Companion plugin provides advanced customization options for the Nony Portfolio theme. It allows you to easily manage your profile badges, social links, navigation, and footer content through an intuitive admin interface.

**Features:**

* Manage profile badges with live preview
* Customize social media links with icons
* Configure navigation and footer content
* Easy-to-use admin interface
* Automatic integration with Nony Portfolio theme

== Installation ==

1. This plugin is automatically installed when you activate the Nony Portfolio theme
2. Alternatively, upload the plugin files to `/wp-content/plugins/nony-portfolio-companion/`
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Portfolio > General Settings to configure your options

== Frequently Asked Questions ==

= Does this plugin work with other themes? =

This plugin is specifically designed for the Nony Portfolio theme. While it may work with other themes, full functionality is only guaranteed with Nony Portfolio.

= How do I add custom social links? =

Go to Portfolio > Social Links in your WordPress admin. You can add, edit, or remove social links. Use Remix Icon classes for icons (e.g., ri-twitter-x-fill, ri-instagram-fill).

= Can I customize the profile badges? =

Yes! Go to Portfolio > Profile Badges to add, edit, or remove badges. You can use emojis and text. Changes are previewed in real-time.
